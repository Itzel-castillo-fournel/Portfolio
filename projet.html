Macro Kénora

Description détaillée

La macro Kénora est un outil développé pour automatiser le processus de consolidation des données issues de multiples fichiers d'export. Ce projet répond à un besoin critique de l'équipe RH qui devait auparavant gérer manuellement la fusion de ces données, un processus chronophage et sujet aux erreurs.

Chronologie du projet

Phase 1 (2 mois): Analyse des besoins et conception de la solution
Phase 2 (3 mois): Développement de la macro et tests
Phase 3 (1 mois): Déploiement et formation des utilisateurs

Fonctionnalités clés

Consolidation automatique des fichiers d'export Kénora

Détection et gestion intelligente des doublons

Validation des données selon des règles métier

Interface utilisateur intuitive

Outils et technologies

VBA Excel

Gestion des avenants chef d'équipe

Description détaillée

Le projet de gestion des avenants chef d'équipe vise à centraliser et optimiser le processus de création et de suivi des avenants au sein du logiciel Pléiades. Cette solution élimine les saisies multiples qui étaient auparavant nécessaires, réduisant ainsi les risques d'erreurs et améliorant l'efficacité du département RH.

Chronologie du projet

Phase 1 (2 mois): Analyse des besoins et spécifications fonctionnelles
Phase 2 (7 mois): Développement de la fonctionnalité dans Pléiades
Phase 3 (1 mois): Implémentation du back-office pour l'édition des avenants et tests

Fonctionnalités clés

Centralisation de la saisie des avenants

Interface de gestion intuitive dans Pléiades

Gestion des niveaux de confidentialité

Traçabilité complète des modifications

Outils et technologies

Pléiades Station de modélisation NGScript BI

Text Adventure

Description détaillée

Le projet Text Adventure est un prototype de jeu textuel interactif développé dans le cadre d'un projet scolaire. Ce jeu permet aux joueurs d'explorer un monde virtuel et d'interagir avec différents éléments uniquement à travers des commandes textuelles, offrant une expérience immersive qui stimule l'imagination.

Chronologie du projet

Phase 1 (1 semaine): Conception du scénario et des mécaniques de jeu
Phase 2 (2 semaines): Développement du moteur de jeu en Python
Phase 3 (1 semaine): Tests et ajustements

Fonctionnalités clés

Système de navigation par commandes textuelles

Gestion d'inventaire et d'objets interactifs

Dialogues avec des personnages non-joueurs

Système de quêtes et d'énigmes

Multiples fins possibles selon les choix du joueur

Outils et technologies

Python Visual Studio Code GitHub

Billeterie

Description détaillée

Le projet Billeterie est une application développée pour calculer automatiquement les tarifs des billets de métro selon diverses règles tarifaires complexes. Cette solution simplifie le processus d'achat pour les utilisateurs tout en assurant l'application correcte des différentes politiques de tarification.

Chronologie du projet

Phase 1 (3 jours): Analyse des règles tarifaires et conception
Phase 2 (1 semaine): Développement du moteur de calcul en Python
Phase 3 (4 jours): Tests et validation

Fonctionnalités clés

Calcul automatique des tarifs selon différents critères

Gestion des réductions et des forfaits

Interface utilisateur simple et efficace

Validation des entrées utilisateur

Génération de reçus détaillés

Outils et technologies

Python Visual Studio Code GitHub
